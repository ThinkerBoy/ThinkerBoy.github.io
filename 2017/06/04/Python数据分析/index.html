list列表:

1、有序的集合，可以按照索引和倒序的方式进行访问；

2、append()总是把新的元素添加到 list 的尾部；insert()方法，它接受两个参数，第一个参数是索引号，第二个参数是待添加的新元素。

3、pop()方法总是删掉list的最后一个元素，并且它返回删除的元素。list.pop(2)删除索引为2的元素

tuple元组：

1、有序的列表，tuple一旦创建完毕，便不能修改

2、( )替代了[ ]，tuple没有 append()、insert()和pop()方法。

3、单元素tuple创建应为：t = (1,).

dict字典:

1、dict的查找速度快，无论dict有10个元素还是10万个元素，查找速度都一样。而list的查找速度随着元素增加而逐渐下降。dict的缺点是占用内存大，还会浪费很多内容，list正好相反，占用内存小，但是查找速度慢。

2、key不能重复。存储的key-value序对是内部无序！

3、dict的key 的元素必须不可变

set：

1、set的元素没有重复，且无序，获得key，而不需要value

2、创建set的方式是调用set（），并传入一个list，list的元素将作为set的元素：

3、由于set存储的是无序集合，所以没法通过索引来访问。访问 set中的某个元素实际上就是判断一个元素是否在set中。

函数式编程

1、高阶函数:

2、map()函数：map()是python内置的高阶函数，接收函数f 和一个list,并通过把函数f依次作用在list的每个元素上，得到一个新的list并返回。

3、reduce()函数：Python内置的高阶函数。reduce()函数接收的参数和 map()类似，一个函数 f，一个list，但行为和 map()不同，reduce()传入的函数 f 必须接收两个参数，reduce()对list的每个元素反复调用函数f，并返回最终结果值。

4、filter函数：Python 内置高阶函数，filter()函数接收一个函数 f 和一个list，这个函数 f 的作用是对每个元素进行判断，返回 True或 False，filter()根据判断结果自动过滤掉不符合条件的元素，返回由符合条件元素组成的新list。

python中笔包

1、像这种内层函数引用了外层函数的变量（参数也算变量），然后返回内层函数的情况，称为闭包（Closure）。

2、闭包的特点是返回的函数还引用了外层函数的局部变量，所以，要正确使用闭包，就要确保引用的局部变量在函数返回后不能变。
