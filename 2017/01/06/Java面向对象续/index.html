基本数据类型:

  • java为8个基本的数据类型提供了对应的包装类，可以使其对应变成对象使用。自动装箱和自动拆箱，允许把基本类型值直接赋给对应的包装类引用变量，也允许把包装类对象直接赋给对应的基本类型变量。基本数据类型变量->包装类实例，是通过对应包装类的构造器来实现的。除了Character之外，还可以通过传入一个字符串参数来构建包装类对象。

    | byte | short | int | long | char | float | double | boolean |
    | —- | —– | ——- | —- | ——— | —– | —— | ——- |
    | Byte | Short | Integer | Long | Character | Float | Double | Boolean |

    ​

  • java利用final关键字来修饰变量、方法和类，系统不允许为final变量重新赋值，子类不允许覆盖父类的final方法，final不允许派生子类。即利用final实现java的不可变类。

  • abstract 和interface 两个关键字分别用于定义抽象类和接口，抽象类和接口都是从多个类中抽象出来的共同特征。抽象类主要作为多个类的模版，接口则定义了多类应用遵守的规范。

  • enum关键字则用于创建枚举类型，枚举类是一种不能自由创建对象的类，枚举类是一种不能自由创建对象的类，枚举里的对象在定义类时已经固定下来。
