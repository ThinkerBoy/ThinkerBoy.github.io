java的三大特征：封装、继承、多态

static:用于修饰方法、属性等成员。所修饰的成员属于类所有，而不是该类的单个实例，所修饰的属性通常 成为类属性、类方法。不适用static修饰的方法、属性则属于该类的单个实例，而不是属于该类。通常把不使用static修饰的属性和方法也称为实例属性、实例方法。静态成员不能直接访问非静态成员。

this:ths关键字是一个对象的默认引用，构造器中引用该构造器执行初始化对象；在方法中引用调用该方法的对象。

区别：对于static修饰的方法而言，则可以使用类来直接调用该方法，如果在static修饰的方法中使用this关键字，则这个关键字则无法指向合适的对象。static修饰的方法中不能使用this关键字，static修饰的方法不能访问不使用static修饰的普通成员。如果要在static修饰的方法中，访问一个普通方法，则需要new 方法名来调用该方法。如果方法里有局部变量和属性同名，但程序又需要在方法里，访问这个被覆盖的属性，则需要使用this前缀。

定义形参长度可变的方法:

长度可变的形参只能处于形参列表的最后。一个方法中最多包含一个长度可变的形参。调用包含一个可变长度形参的方法时，这个长度可变的形参即可以传入多个参数，也可以传入一个数组。

  • 方法重载:同一个类中的方法名相同，参数列表不同。

    成员变量和局部变量:

    • 成员变量:实例属性(不以static修饰)，类属性(以static修饰)
    • 局部变量：形参(方法签名中定义的变量)，方法局部变量(方法内定义)，代码块局部变量 (在代码块内定义)
    • 区别与联系：类属性为类成员变量，为类的成员，与类共存亡；实例属性为实例成员变量,与实例共存亡。访问形式：类.类属性，实例.实例属性，实例.类属性。局部变量定义时，必须显示初始化之后才能使用，系统不会为局部变量进行初始化。

    隐藏和封装:将对象的状态信息隐藏在对象内部，不允许外部程序直接访问对象内部信息，而是通过该类所提供的方法来实现对内部信息的操作和访问。

    private:修饰，则只能在该类的内部被访问，常用来修饰属性，将其隐藏在类的内部。

    default：被相同包下的其它类访问。

    protected：同一个包里的类访问，也可以被不同包中的子类访问。常用来子类重写父类的某个方法。

    public:可以被所有类访问

    类的继承:通过extends关键字来实现。子类包含与父类同名方法的现象被称为方法重写，即方法覆盖(Override)。super和this都不能出现在static修饰的方法中。继承是从最顶层开始执行。

    多态性:java引用变量有两个类型：一个是编译时的类型，一个是运行时的类型，编译时的类型由声明该变量时使用的类型决定，运行时类型由实际赋给该变量的对象决定。如果编译和运行时的类型不一致，则就会出现多态（Polymorphism）.父类引用指向子类对象，编译时类型与运行时类型不一致。编译时是类型是父类，运行时类型是子类，执行子类覆盖父类后的方法；当父类无某个方法，而子类有时，运用多态将会报错。因为编译类型时为父类，编译时无法调用该方法。(以上均指的是父类引用指向子类对象)。属性不具有多态特性，仍然调用的是父类的属性。引用变量只能调用它编译时的方法，而不能调用运行时类型的方法。可以运用强制类型转换，来调用运行时方法。

    Instanceof:（ 引用类型变量 instanceof 类）用于判断前面的对象是否是后面的类或者其子类，实现类的实例。如果是，则返回true,否则返回 false。instanceof 运算符前面操作数的编译时类型要么与后面的类相同，要么是后面类型的子类，否则会引起编译错误。

    继承与组合:继承实现类的重用，但是破坏了类的封装特性；组合同样是实现类的重用，可以提供更好的封装。

    设计父类应遵循如下原则：

    • 尽量隐藏父类的内部数据，将其所有属性设置为：private，禁止子类直接访问父类的属性。
    • 不让子类随意访问和修改父类的方法。父类中辅助工具的方法，应为private；父类中需要被外部类调用，必须以public修饰，但又不希望子类重写该方法，则可以使用final关键字。如果希望父类的某个方法被重写，但是不希望其它类自由访问，使用protected关键字修饰该方法。
    • 不用在父类构造器中调用被子类重写的方法。即当子类重写了父类的某个方法，则不要在父类的构造方法中调用被重写的该方法。

    组合:把一个类当成另一个类的组成成分，从而允许新类直接复用该类的public方法。继承是直接访问子类从父类那里继承的方法。组合：把旧类对象作为新类的属性嵌入，用以实现新类的功能，用户看到的是新类的方法，而不能看到嵌入对象的方法。通常需要用private修饰嵌入的旧类对象。

    继承要表达的是"is-a"的关系，而组合表达的是"has-a"的关系。

    初始化块:java常使用构造器来对单个对象进行初始化操作，初始化块的修饰符只能是static，使用萨static修饰的初始化，是类相关的，系统将在类初始化时执行静态初始化块，而不是在创建对象时执行，静态初始化块比普通初始化块先执行。java创建对象时，系统总是优先调用该类里定义的初始化块，然后再执行构造函数。
